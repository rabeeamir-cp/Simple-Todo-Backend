# Todo MVC API (Node + Express + Postgres)

Simple Todo CRUD API using MVC structure with Postgres and a basic migration.

## Tech Stack

- Node.js
- Express
- PostgreSQL
- `pg` (node-postgres)
- Simple SQL migration (no heavy ORM)

---

## 1. Prerequisites

- Node.js (LTS)
- npm or yarn
- PostgreSQL running locally or accessible via connection string

---

## 2. Setup on your PC

1. Clone or copy this project to your machine.
2. Open a terminal in the project root.
3. Install dependencies:

   ```bash
   npm install
